React Native - Picker

选择框

  • selectedValue 当前选中值

  • mode Android Only 展现方式

    1. dialog:对话框形式
    2. dropdown:下拉框形式

  • prompt Android Only 当以对话框形式显示时的提示标题

  • onValueChange 当选项变化时的回调

    1. itemValue:当前选中的值
    2. itemPosition:当前选中的项的索引
